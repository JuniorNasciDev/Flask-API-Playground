# Flask-API-Playground
esse repositório são diversas apis desenvolvidas em python com flesk, para meus estudos sobre api's

Basicamente a ideia é seguir essa lista de exercicio e adquirir essas habilidades descritas 

---------------------------Play Ground-------------------------------------------


===========================================================
1. "Contador de Palavras Simples"
Descrição:
Crie uma API que recebe uma frase e retorna o número de palavras que ela contém. O parâmetro será enviado pela URL.

Habilidades adquiridas:
- Manipulação de strings em Python
- Trabalhar com parâmetros de URL no Flask
- Contagem de elementos em uma string

Assunto indicado a estudar para realizar o exercício:
- Manipulação de strings em Python
- Flask básico: rotas e parâmetros
- Loops e condicionais em Python

===========================================================
2. "Calculadora Simples"
